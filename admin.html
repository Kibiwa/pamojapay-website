📚 How This CMS Works

This admin panel saves all your website content to your browser's local storage. When you save changes, the content is stored locally and the main index.html reads it on load.

To make changes permanent: After saving here, click “Export Content JSON” in Settings, then paste that JSON into your content.json file in your GitHub repo. Vercel will auto-deploy.

Hosting recommendation: You are already set up perfectly. Keep Vercel + GitHub. See the Setup Guide in Settings.

🚀 Hosting Setup Guide

Your current setup: GitHub + Vercel ✓

This is the best free hosting stack for a static site like this. Here is how your workflow works:

  1. You edit content here in the Admin CMS
  2. Click "Save Changes" → stored in browser localStorage immediately
  3. Click "Export content.json" below → downloads a JSON file
  4. In your GitHub repo, replace content.json with the new file
  5. Vercel auto-deploys within ~30 seconds
  6. Live site now shows your updated content

⚠ Admin panel access: Host admin.html in the same Vercel project. Protect it by adding a Vercel password or just keeping the URL private.
